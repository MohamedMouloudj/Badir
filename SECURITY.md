# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please **DO NOT** create a public issue.

Instead, please report it privately by:

- Using GitHub's private vulnerability reporting (see Security tab)
- Or emailing us at: atm.website.test@gmail.com

We will respond within 48-72 hours and work with you to address the issue.

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 1.x     | ✅        |
| < 1.0   | ❌        |
